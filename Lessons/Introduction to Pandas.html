Pandas Overview¶

Over the next couple of series, we are going to go through an overview of Pandas, a data manipulation package for python. Pandas has extensive, easy to use data manipulation capabilities that are useful for data transformation, preparation and analysis. Let's begin.

Import Pandas¶

Convention is a powerful thing and you will notice that all pandas users import pandas the same way.

In [1]:
# Importing Pandas
import pandas as pd

# Checking Pandas Version
pd.__version__
Out[1]:
'0.20.1'

Reading Data into a pandas object¶

Pandas allows us to read data in various formats and from various sources. General, people read in from a file and the only requirement is to pass the file path into the read methods available in pandas. Below is an example of reading from csv file.

In [2]:
# Reading the Hotel's Review CSV file
data = pd.read_csv(r'/Volumes/Sifael/DS Templates/Data/titanic.csv')

drop¶

Pandas allows us to drop both rows and columns as need. We use the drop to pass a list of columns we want to drop from the dataframe.

Note:
axis=1: Implements drop by for column index names that match the list provided
axis=0 : Implements drop by for row index names that match the list provided
inplace=True: Implements the drop on the current dataframe object

data.drop(['PassengerId', 'Parch'], axis=1, inplace=True)

3. Statistics¶

Pandas comes with statistical function that are straight forward and easy to use. Let's look at these below:

Mean Absolute Deviation - mad¶

Mathamatically, the MAD is expressed as: $$mad = \frac {1}{N} \sum |x_i - \bar x|$$

We can also compute the Mean Absolute Deviation with the mad method.

In [23]:

Out[23]:

Variance - var¶

Variance is the measure of variability of the data from its central tendency. Mathematically, we express variances as:

$$var(x) = \sum (x_i - \bar x)^2$$

Computing the variance with the var method.

In [24]:
data.var()
Out[24]:
Survived       0.236772
Pclass         0.699015
Age          211.019125
SibSp          1.216043
Fare        2469.436846
dtype: float64

Standard Deviation - std¶

Standard Deviation is the standardized measure of how variant the data is from its mean. Mathematically, it is expressed as:

$$s = \sqrt{\frac {\sum (x - \bar x)^2 }{N-1} } $$

In pandas, we can calculate the standard deviation as follows:

In [25]:
data.std()
Out[25]:
Survived     0.486592
Pclass       0.836071
Age         14.526497
SibSp        1.102743
Fare        49.693429
dtype: float64

Mean¶

The mean (expected value) is the sum of all occurrences divided by the count of all occurrences. Mathematically, we express it as:

$$mean(x) = \frac {1}{N} \sum x_i$$

Computing the Mean:

In [26]:
data.mean()
Out[26]:
Survived     0.383838
Pclass       2.308642
Age         29.699118
SibSp        0.523008

dtype: float64

Correlation¶

Mathematically, correlation is the covariance of two variables divided by their standard deviations. It is expressed as follows:

$$\rho_{X,Y} = \frac {cov(X,Y)} {\sigma_x \sigma_y}$$
